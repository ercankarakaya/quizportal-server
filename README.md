# quizportal-server
This is a quiz application.

# Pre-requirements:

### Java 11
### Spring Boot v2.6.0
### Database: PostgreSQL
### Hibernate with JPA
### JWT Authentication For Security
### Aspect For Logging & Exceptions
### Lombok Extension

- You should enable Lombok extension in your IDE.
