# quizportal-server
Quiz Project Backend. Tools: Spring Boot, Rest API, JPA, Oracle Database.

# Pre-requirements:

### Java 1.8 or 11
### Spring Boot v2.6.0
### Database: Oracle 18c XE
### Hibernate with JPA
### Lombok Extension

- You should enable Lombok extension in your IDE.
